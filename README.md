Dashboard-Extended
==================

Dashboard for live monitoring of distributed components


## Installation

Pre-requisite:

1) browserify : ~3.31.2
2) bower : ~1.2..8
3) redis
4) nodemon : 1.0.15
5) sass

```
gem install sass
```




Resolve local Dependencies
==============

```
npm install
```


### Generate client side script

```
npm run-script browserify
```


## Development

Run Server

```
npm start

```
